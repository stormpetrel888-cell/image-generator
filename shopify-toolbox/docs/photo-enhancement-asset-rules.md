# Photo Enhancement Asset Rules

Purpose: prevent confusion between separate photos, true enhancement, and AI art transformation.

## Core rule
Do not label two different photos as before/after.

## Asset types

### Original / Input
A customer-owned or user-owned photo before editing.

### True Enhancement
Same photo, same composition, same place, same subject.
Allowed fixes:
- noise reduction
- sharpness
- color correction
- contrast
- exposure
- crop cleanup
- print preparation

Not allowed for true enhancement:
- replacing the sky
- replacing the scene
- removing major real elements unless requested
- turning it into a different place

### AI Art Transformation
A new artwork inspired by the original photo.
Allowed changes:
- more dramatic sky
- changed composition
- stylized mood
- artistic expansion
- surreal or cinematic version

## Shopify wording rule
If the same photo is preserved, call it photo enhancement.
If the image becomes a new artwork, call it AI art transformation or photo-inspired art.

## Current lesson
The dramatic sky image and the tree-line sunset image are separate assets, not before/after of the same photo.
