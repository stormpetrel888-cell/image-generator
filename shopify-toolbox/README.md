# Shopify Toolbox

Purpose: small reusable Shopify workflow tools for DealzMart, Miss Major, and Solaryn Grey.

Rule: one tool, one job, one test.

## Current first tool
Product Listing Builder

Goal: turn rough product/service information into Shopify-ready listing copy.

## Folder map

- product-listings/ = product listing builder templates and outputs
- image-assets/ = image naming and asset workflow notes
- seo/ = SEO title, meta description, tags, and handle rules
- checklists/ = store setup and listing verification checklists
- docs/ = usage notes and service model notes
- tests/ = before/after proof tests

## Operating rule
Build small. Test on our own stores first. Keep winners, kill junk.
