# Miss Major Video Tools

Small local helpers for making slot videos faster without breaking authenticity.

## Goal

Real slot play stays real. Tools only add timing notes, text gags, reaction cutaways, and quick B-roll helpers.

## First useful tool idea

`screenshot_cutaway_builder.py`

Input:
- screenshot or freeze-frame from slot footage
- short gag line
- duration, usually 4-6 seconds

Output:
- MP4 cutaway ready to drop into Filmora

## Rules

- Do not hide bankroll movement.
- Do not fake wins.
- Return to the next real spin after a gag.
- Use comedy cutaways only to help dead stretches, near misses, or funny moments.
- Keep it fast enough to use, not fancy enough to become a second job.

## Possible cutaway types

- machine talks back: "I HEARD THAT"
- near miss tease: "HA HA... ALMOST"
- dead spin loading bar: "Searching for payout... no results found"
- Miss Major reaction: sleeping, dancing, glaring, shrugging
- tiny text popups: lunch money, rude little machine, don't threaten me with hope
